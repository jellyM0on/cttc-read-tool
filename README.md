### Epub Reader (v.0.1 UI Prototype)

Offline-capable document reader designed specifically for language learning. This application transforms EPUB content into readable HTML and enables persistent highlighting, helping users actively engage with texts, annotate vocabulary, and build deeper comprehension.

This version is the UI Prototype and does not have fully functional features yet (i.e. Cloud Support)

### Features
- 📖 EPUB Reader
  - The reader parses the epub file and renders it to plain html. This ensures compatibility with external tools (i.e. extension-based translators)
  - Customizable font style (including Dyslexic-friendly fonts), font size, and background themes
- 🗂️ Library System
  - Upload and store documents locally
- ✍️ Highlighting System
  - Create and manage highlights inside documents
  - Highlights persist using IndexedDB
  - Dedicated highlights panel for easy access
- 💾 Offline Support
  - Documents stored locally via IndexedDB
  - No backend required for core functionality

### Stack
- React TS, Tailwind
- IndexedDB
